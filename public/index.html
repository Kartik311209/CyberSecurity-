IoPA — Indicators of Pre-Attack Activity

Check all observed indicators before the attack occurs to calculate pre-attack risk:

  • Unusual login times (outside business hours 09:00–18:00)
  • Multiple failed login attempts (<5 min window)
  • Login from new or unknown geographic location
  • Rapid sequential port scanning detected
  • Abnormal outbound network traffic spike (>3σ)
  • Access to sensitive files outside normal role scope
  • New privileged user accounts created unexpectedly
  • Privilege escalation attempts (sudo / SYSTEM token)
  • Known malicious IP observed in firewall logs
  • Security tools or audit logs disabled or modified
